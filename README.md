# create-weekend-app

Scaffold a Next.js 15+ project with a "batteries-included" Weekend Stack.

## Features

- **Next.js 15+** (App Router, TypeScript, Tailwind)
- **Zustand** - Global state management
- **@tanstack/react-query** - Async data fetching
- **Framer Motion** - Smooth animations
- **Lucide React** - Beautiful icons
- **Recharts** - Responsive charts
- **Sonner** - Elegant toast notifications

## Usage

```bash
npx create-weekend-app my-project
```

## What's Included?

The tool automatically configures a `WeekendProvider` to wrap your app with necessary providers and injects a high-quality "Counter & Analytics" dashboard in `src/app/page.tsx` to get you started immediately.
